## 🚀 Introduction

In this tutorial, you'll learn how to build a modern portfolio website using **React**, **TailwindCSS**, **Vite**, and **Lucide Icons**. From dark mode support to responsive animations and deployable project showcases, this guide walks you through every step—perfect for developers looking to level up their frontend skills or showcase their work.

🎥 Watch the full tutorial here: [CodeWithKeya on YouTube](https://www.youtube.com/@CodeWithKeya)

---

## ⚙️ Tech Stack

* **React** – Component-based UI development  
* **Vite** – Lightning-fast build tool  
* **TailwindCSS** – Utility-first CSS for styling  
* **Lucide Icons** – Clean and beautiful icon pack  
* **Radix UI** – Accessible component primitives  
* **TypeScript (optional)** – Type safety and tooling  
* **GitHub & Vercel** – Deployment  

---

## ⚡️ Features

* 🌑 **Light/Dark Mode Toggle**  
  Save theme preference in local storage with smooth transitions  

* 💫 **Animated Backgrounds**  
  Stars, meteors, scroll effects, and glowing UI elements  

* 📱 **Responsive Navigation**  
  Desktop and mobile menus with glassmorphism  

* 👨‍💻 **Hero & About Sections**  
  Showcase your personality with smooth intro animations and call-to-action buttons  

* 📊 **Skills Grid**  
  Filterable progress bars and skill categories with animated widths  

* 🖼️ **Projects Showcase**  
  Display project screenshots, tech stacks, and live/demo GitHub links  

* 📩 **Contact Section**  
  Social icons plus a responsive contact form with toast notifications  

* 🚀 **One-Click Deployment**  
  Easily host your portfolio site using Vercel and GitHub integration
